
# 文档解析和转换

功能：
1. 解析常用office文件成markdown，比如word、excel、ppt
2. 解析image,PDF成markdown
3. 将markdown转换成word或pdf

使用工具
1. markitdown
2. MinerU
3. Qwen2.5-VL
4. Pandoc
